# Mechanical Armor Lottery

## Overview
A high-tech, mechanical-themed lottery webpage featuring:
- **Procedural Animations**: Smooth CSS3 transitions and animations.
- **Synthesized Audio**: Built-in sound effects using Web Audio API (no external files needed).
- **Responsive Design**: Works on Desktop and Mobile.
- **Tailwind CSS**: For utility-first styling.

## How to Run
Simply double-click `index.html` to open it in your browser.

## Customization
- **Prizes**: Edit the `PRIZE_TYPES` array in `script.js`.
- **Slots**: Change `SLOT_COUNT` in `script.js`.
- **Styles**: Adjust colors in `index.html` (Tailwind config) or `style.css`.
